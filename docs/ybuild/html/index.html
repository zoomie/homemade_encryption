homemade_steganog¶

class main.Steg[source]¶

The class Steg encodes text data into an image. The text is transformed into numbers, these numbers are then inserted into specific image pixel locations.

decrypt_img(img, custom_indexes=None)[source]¶

Extracts text from the image.

encrypt(custom_indexes=None)[source]¶

The encrypt method encodes the data into the image by changing pixels value, the index is chosen by the num_generator.

load_data(data_path, ending_numbers=None)[source]¶

Loads the data and converts it into digits.

load_img(img_path)[source]¶

Loads the image.

Purpose¶

The purpose of steganography is concealing particular data in normal data. In this module, the particular data is a message while the normal data is an image.

As can be seen in the simple example below, the message can be hidden in the image using the encrypt method and removed at its endpoint using the decrypt method.

Simple example¶

Sending a text file hidden in an image:

from homemade_steganog import Steg

img_path = '../data/img.jpeg'
data_path = '../data/text.txt'
s = Steg()
s.load_img(img_path)
s.load_data(data_path)
data_in_img = s.encrypt()

from matplotlib.pyplot as plt
plt.imshow(data_in_img)
plt.show()

On the second users machine:

from homemade_steganog import Steg
s_new = Steg()
# Recieve data_in_img
message = s_new.decrypt_img(data_in_img)

Custom example¶

Using custom_indexs specifies the locations (pixel indexes) where each digit of the message is hidden:

img_path = '../data/img.jpeg'
data_path = '../data/text.txt'
s = Steg()
s.load_img(img_path)
s.load_data(data_path)
custom_indexs = [145, 82, 39, 94, 109]
data_in_img = s.encrypt(custom_indexs)

# Send image here
plt.imshow(data_in_img)
plt.show()

s_new = Steg()
custom_indexs = [145, 82, 39, 94, 109]
message = s_new.decrypt_img(data_in_img, custom_indexs)
print(message)
